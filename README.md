# Budget Flow

一个更适合纽约日常消费习惯的轻量记账 App，适合在 iPhone 浏览器打开，也可以添加到主屏幕。

## 现在已经有的功能

- 记录收入和支出
- 按月份查看收支概览
- 按支出类型统计占比
- 根据当前账本给出更贴近纽约生活的开源节流建议
- 使用美元金额和更符合美国日常的支付方式、分类
- 上传小票图片，在浏览器里本地 OCR 识别后入账
- 数据默认保存在浏览器本地 `localStorage`

## 怎么打开

这是一个纯前端静态项目。

### 方式 1：直接打开

直接双击 `index.html` 就能看到页面。

### 方式 2：用本地静态服务器打开

如果你本机装了 Python，可以在这个目录里运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## iPhone 上怎么用

1. 把这个项目放到一个你能在 iPhone 打开的地址上。
2. 用 Safari 打开页面。
3. 点击“分享”。
4. 选择“添加到主屏幕”。

这样它会像 App 一样全屏打开。

## 小票识别说明

- 选择一张 receipt 图片。
- 点击“开始识别”。
- 识别结果会自动填到表单里，你确认后再保存。
- 第一次识别时浏览器可能会下载 OCR 模型文件，所以会稍慢一点。

## 当前限制

- 目前是本地版，数据没有云同步。
- 本地 OCR 识别准确率通常不如大模型，复杂小票建议手动检查。
- 第一次使用 OCR 时通常需要联网下载识别模型。
